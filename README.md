# VSmart Match

Monorepo for VSmart Match, an AI-powered resume parsing and candidate-job matching platform.

## Structure

- [`vsmart-backend/`](vsmart-backend) — FastAPI backend: resume parsing, candidate-job matching, and developer profile aggregation (GitHub, LeetCode). See its [README](vsmart-backend/README.md) for setup and architecture.
- [`vsmartmatch-frontend/`](vsmartmatch-frontend) — Next.js frontend. See its [README](vsmartmatch-frontend/README.md) for setup.

Each subdirectory retains its full original commit history, merged in via `git subtree`.
